### Super Administrador
- [ ] Altas de Parque
- [ ] Modificaciones de Parque
- [ ] Consultas de Parque

       Ver sistema para mas detalles.

- [ ] Modificaciones de Empresa 
- [ ] Consultas de Empresas
     * Despliega listado de todas las empresas

       Ver sistema para mas detalles.
